  • Tipos de bibliotecas:
    • Estandar:
      bibliotecas oficiales de python.
    • Externas:
      La forma más común de instalar paquetes o bibliotecas externas en python es usar el instalador PIP.
      pip install "biblioteca"

      biblioteca de cómputo numérico de python (arreglos) numpy.
      pip install numpy

      biblioteca para análisis de datos (tablas) pandas.
      pip install pandas

      biblioteca para crear gráficas (barras, pie, series, etc...) matplotlib.
      pip install -U matplotlib

  • Problemas:
  • Un programa donde uses la biblioteca de random para llenar una lista con números aleatorios y después la muestre en pantalla.
  • Un programa donde uses la biblioteca de numpy para convertir la lista en un arreglo con números aleatorios y después lo muestre en pantalla.
  • Un programa donde uses la biblioteca de matplotlib para que a partir de 2 arreglos con 10 números cada uno, cree una gráfica de tipo scatterplot.
